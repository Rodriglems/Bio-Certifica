
  # App de Agroecologia

  This is a code bundle for App de Agroecologia. The original project is available at https://www.figma.com/design/Nmk4CNSVMGt1dAKvWVwv2X/App-de-Agroecologia.

  ## Running the code

  Run `npm i` to install the dependencies.

  Create a `.env` file in the project root with at least:

  `DATABASE_URL=postgres://USER:PASSWORD@HOST:5432/DBNAME`

  Run database migrations before first use:

  `npm run db:migrate`

  Run `npm run dev` to start the development servers (Vite + API).
